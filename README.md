# Overview
The E-Commerce Slight Dashboard is an interactive Power BI dashboard built to analyze and visualize key e-commerce business metrics. It provides a comprehensive view of sales performance, customer activity, product trends, payment methods, and country-wise sales through dynamic and easy-to-understand visualizations. The dashboard enables users to monitor business performance, identify growth opportunities, and make informed, data-driven decisions using interactive filters and real-time insights.
# Features
- 📅 Month-wise Sales Filtering
- 🛍️ Category-wise Product Analysis
- 👥 Customer Overview
- 📦 Total Orders Tracking
- 💰 Total Sales Analysis
- 📈 Average Order Value (AOV)
- 💳 Payment Method Distribution
- 🌍 Country-wise Sales Map
- 📊 Monthly Sales Trend
- 📋 Product Performance Table
- 📈 Daily Sales Trend (Sparklines)
- 📉 Growth Percentage Analysis
